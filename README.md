# Crypto Desktop App

This is a code repository for the cryptocurrency desktop application

## Quick Start

1. `git clone git@github.com:zero0rder/crypto-desktop.git`
2. `cd crypto-desktop`
3. `yarn install`
4. `yarn start`

* React
